Place your Vosk model files here.

Download from: https://alphacephei.com/vosk/models

Recommended models:
  - vosk-model-small-hi-0.22  (Hindi, works for Marathi digits/numbers)
  - vosk-model-mr-0.8         (Marathi specific, if available)

After downloading, extract the zip so this folder contains:
  vosk_model/
    am/
    conf/
    graph/
    ivector/
    ...

If no model is placed here, the app will fall back to keyboard input automatically.
